## Установка и запуск

- `npm install` — установка
- `npm run dev` — запуск для разработки с вотчером
- `npm run start` — запуск с Express
- `npm run build` — сборка стабильной версии

## Спринт 1

### Использование шаблонизатора и компонентного подхода
